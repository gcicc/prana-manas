================================================================================

[✓] Meditation Timer
    - Circular progress ring display with large time counter
    - Preset durations: 5, 10, 15, 20, 30, 45, 60 minutes
    - Custom duration input
    - Immersive full-screen mode

[✓] Audio System
    - Start bell (262Hz singing bowl tone with decay)

HOW TO USE
================================================================================

1. Visit https://gcicc.github.io/prana-manas/ on mobile or desktop

2. Add to home screen:
   iOS: Tap Share → Add to Home Screen
   Android: Tap menu → Install

3. Start meditating:
   - Select duration (or set custom)
   - Choose ambient sound (optional)
   - Enable "Guided Prompts" for metta practice
   - Tap "Start Meditation"
   - Screen stays on, immersive mode activates
   - Bell sounds mark time intervals
   - After completion, rate mood and optional note
   - Session auto-saved to history

4. Track progress:
   - Go to "Log" tab to see stats, calendar, history
   - Monitor streaks and total minutes

5. Explore more:
   - "Hub" tab links to other Prana Suite apps
